## Step 1: Google Search Console Setup

### 1.1 Add Property
1. Go to **Google Search Console**: https://search.google.com/search-console
2. Click **"Add property"**
3. Enter your site URL: `https://ahmed-coding.vercel.app`
4. Choose **URL prefix** method (recommended for Vercel)

### 1.2 Verify Domain Ownership
Select one of these verification methods:
- **Recommended**: HTML file upload (simplest with Vercel)
- **Alternative**: HTML tag (add meta tag to index.html)
- TXT record (DNS verification)

**To verify with Vercel:**
1. Download the HTML verification file from Google
2. Place it in `/public/` folder
3. Deploy to Vercel
4. Click "Verify" in Google Search Console

### 1.3 Submit Sitemap
1. In GSC, go to **Sitemaps** section
2. Enter: `https://ahmed-coding.vercel.app/sitemap.xml`
3. Click **Submit**

### 1.4 Monitor Performance
- Check **Performance** tab for clicks, impressions, CTR
- Review **Coverage** to see indexed pages
- Monitor **Core Web Vitals** in Enhancements section

## Step 2: Bing Webmaster Tools Setup

### 2.1 Add & Verify Site
1. Go to **Bing Webmaster Tools**: https://www.bing.com/webmaster
2. Click **Add site** → Enter `https://ahmed-coding.vercel.app`
3. Verify using Google Search Console (auto-import available)

### 2.2 Submit Sitemap
1. Go to **Sitemaps** section
2. Add: `https://ahmed-coding.vercel.app/sitemap.xml`
3. Submit

### 2.3 Configure Crawl Settings
- Set crawl control to "Normal"
- Review robots.txt compliance
- Check for crawl errors
